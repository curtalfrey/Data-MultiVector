Usage Instructions

After you have successfully installed Data-MultiVector, here's how you can use it:

  1. Access the Application:
    Open a web browser and navigate to http://localhost:5000 to access Data-MultiVector.
    For subdomain access, set up the necessary DNS and server configurations.
  2. Interact with the Application:
    • Inputs:
      Use the following format for defining input paths: 
      Input-A:
      Input-B:
      Input-C:
    • Outputs:
      Use the following format for defining output paths:
