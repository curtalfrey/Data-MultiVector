Data-MultiVector Documentation

Installation Instructions (Docker)

  1. Clone the Repository:
    Open a terminal on your server and navigate to your desired deployment directory.
    Run the following command to clone the repository: 
    git clone https://github.com/curtalfrey/Data-MultiVector.git /home/user/github/Data-MultiVector
  2. Navigate to the Project Directory:
    Change into the cloned repository directory: 
    cd /home/user/github/Data-MultiVector
  3. Build Docker Image:
    Run the following command to build the Docker image: 
    docker-compose build
  4. Configure docker-compose.yml:
    Open the docker-compose.yml file and update the environment variables under the app service with your desired values.
    Make sure to adjust the INPUT_PATHS and OUTPUT_PATHS according to the directories you want to link.
  5. Start Docker Containers:
    Run the following command to start the Docker containers: 
    docker-compose up -d

Usage Instructions

After you have successfully installed Data-MultiVector, here's how you can use it:

  1. Access the Application:
    Open a web browser and navigate to http://localhost:5000 to access Data-MultiVector.
    For subdomain access, set up the necessary DNS and server configurations.
  2. Interact with the Application:
    • Input Paths:
      Use the following format for defining input paths: 
      Input-A:
      Input-B:
      Input-C:
    • Output Paths:
      Use the following format for defining output paths: 
      Output-1:
      Output-2:
      Output-3:
  3. Create Connections:
    • Link-Alpha:
      Connect Input-A to Output-1 using the following command: 
      link Input-A to Output-1
    • Link-Bravo:
      Connect Input-B to Output-2 using the following command: 
      link Input-B to Output-2
