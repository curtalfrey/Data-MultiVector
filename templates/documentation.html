Usage Instructions

After you have successfully installed Data-MultiVector, here's how you can use it:

  1. Access the Application:
    Open a web browser and navigate to http://localhost:5000 to access Data-MultiVector.
    For subdomain access, set up the necessary DNS and server configurations.
  2. Interact with the Application:
    • Input Paths:
      On the homepage, you will find input fields for defining input paths. Enter the paths in the following format: 
      Input-A:
      Input-B:
      Input-C:
    • Output Paths:
      On the homepage, you will find input fields for defining output paths. Enter the paths in the following format: 
      Output-1:
      Output-2:
      Output-3:
  3. Create Connections:
    • Link-Alpha:
      On the homepage, you will find checkboxes for creating connections. Check the box for "Link-Alpha" to connect Input-A to Output-1.
    • Link-Bravo:
      On the homepage, check the box for "Link-Bravo" to connect Input-B to Output-2.
