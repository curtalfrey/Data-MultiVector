Data-MultiVector Documentation

Installation Instructions (Docker)

  1. Clone the repository from GitHub.
  2. Configure environment variables in the Docker Compose file, including:
    • SECRET_KEY: Your secret key for Flask.
    • SQLALCHEMY_DATABASE_URI: Your MySQL database connection URL.
    • MAIL_SERVER, MAIL_PORT, MAIL_USERNAME, MAIL_PASSWORD: SMTP email server configuration.
  3. Run the Docker Compose command to deploy the application.
  4. Access the application via the configured domain name or IP address.

Usage Instructions

After you have successfully installed Data-MultiVector, here's how you can use it:

  1. Access the Application:
    Open a web browser and navigate to the application's URL to access Data-MultiVector.
  2. Register as a User:
    Click the "Register" link on the homepage to create a user account with your desired username and password.
  3. Login:
    Use your registered username and password to log in to the application.
  4. Interact with the Application:
    • Input Paths:
      On the homepage, you will find input fields for defining input paths. Enter the paths as needed.
    • Output Paths:
      On the homepage, you will find input fields for defining output paths. Enter the paths as needed.
    • Create Connections:
      On the homepage, check the checkboxes to create connections between input and output paths.
    • Delete Links:
      On the dashboard, you can delete connections and choose to keep or delete associated files. Deleted files are moved to your personal trash can.
  5. User Dashboard:
    Once logged in, you'll be redirected to your dashboard where you can manage your connections, trash, and settings.
  6. Logout:
    Click the "Logout" link in the dashboard to log out of your account.
